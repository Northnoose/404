Grunnleggende Python

Introduksjon til Python

Python er et populært, allsidig og lettlest programmeringsspråk som ofte brukes til alt fra webutvikling og datavitenskap til scripting og automatisering. I denne korte leksjonen skal vi se på noen grunnleggende konsepter som:

  • Hvordan man skriver en enkel print-setning
  • Variabler og datatyper
  • Enkle funksjoner

1. Hello World!

I Python kan du skrive: 

print("Hello World!")
for å vise tekst i konsollen.

2. Variabler

En variabel opprettes automatisk når du tildeler en verdi: 

x = 5
y = "Hello"
print(x, y)

3. Enkle funksjoner

Funksjoner defineres med def-nøkkelordet: 

def min_funksjon():
    print("Dette er en funksjon!")
